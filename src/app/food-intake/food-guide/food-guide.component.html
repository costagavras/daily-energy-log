• Meals can be added through USDA database or app's own database.

  • A new user starts with an empty app database which can be populated by copying USDA items or adding new items manually.

  • Manage food items page allows to add, edit or delete food items from the app's database.

  • Food portions can be added in grams or ounces.

  • Inserted meals will appear in the table below input section. Changing the date allows to see, add or delete meals in the past.

  • The log page displays past meals grouped by date. It's possible to toggle the date detail, filter by name and show/hide columns.
